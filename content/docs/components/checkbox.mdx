Renders an `input` with `type="checkbox"`. Can be used as a [controlled or uncontrolled](https://reactjs.org/docs/uncontrolled-components.html) 
component.

```jsx
<Checkbox />
```

<HeadingLink
    as="h2"
    slug="icon"
>
    <Code>icon</Code>
</HeadingLink>

The `icon` prop replaces the icon rendered by the component.

```jsx
<Checkbox
    fontSize="2rem"
    borderWidth="2px"
    borderColor="Primary"
    icon={
        <DotIcon fill="Primary" padding={1} />
    }
/>
```
