Renders an `input` with `type="radio"`. Can be used as a [controlled or uncontrolled](https://reactjs.org/docs/uncontrolled-components.html) 
component.

```jsx
<Radio />
```

<HeadingLink
    as="h2"
    slug="icon"
>
    <Code>icon</Code>
</HeadingLink>

The `icon` prop replaces the icon rendered by the component.

```jsx
<Radio
    fontSize="2rem"
    borderWidth="2px"
    borderColor="Primary"
    icon={
        <CheckIcon fill="Primary" padding={1} />
    }
/>
```
