Renders an `input` with `type="range"`

```jsx
<Slider
    width="200px"
    height="10px"
/>
```

```jsx
<Slider
    appearance="none"
    width="200px"
    height="4px"
    borderColor="Primary"
    backgroundColor="Secondary"
    min="0"
    max="100"
    step="10"
/>
```
