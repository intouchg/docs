The full list of hooks in the [`@intouchg/hooks` package](https://www.npmjs.com/package/@intouchg/hooks) 
can be found below.

* [useCookie](/docs/hooks/useCookie) - store and retrieve cookies and react to changes
* [useInternetListener](/docs/hooks/useInternetListener) - react to changes in internet connectivity
* [useInView](/docs/hooks/useInView) - watch an element's position in or out of the view with `IntersectionObserver`
* [useIsoLayoutEffect](/docs/hooks/useIsoLayoutEffect) - isomorphic `useLayoutEffect` for Server-Side Rendering
* [useMeasure](/docs/hooks/useMeasure) - watch an element's size and position with `ResizeObserver`
* [useMediaQuery](/docs/hooks/useMediaQuery) - watch for matches to a CSS `min-width` media query 
* [useOutsideClickListener](/docs/hooks/useOutsideClickListener) - react to clicks outside an element
* [usePrevious](/docs/hooks/usePrevious) - store and access a value from the previous render
* [useScrollLock](/docs/hooks/useScrollLock) - toggle locking the browser window to prevent scrolling
* [useWindowSize](/docs/hooks/useWindowSize) - watch the window's size


* [useRafMeasure](/docs/hooks/useRafMeasure) - a `useMeasure` hook that only renders on `requestAnimationFrame`s
* [useRafState](/docs/hooks/useRafState) - a `useState` hook that only renders on `requestAnimationFrame`s
* [useRafWindowSize](/docs/hooks/useRafWindowSize) - a `useWindowSize` hook that only renders on `requestAnimationFrame`s
