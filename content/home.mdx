<IntroParagraph paddingTop="0">
    Themeable React component primitives focused on developer experience, performance, and accessibility.
</IntroParagraph>

```jsx
<Box
    color="white"
    backgroundColor="#5e00ff"
    padding="48px"
    textTransform="uppercase"
>
    Hello world
</Box>
```

<IntroParagraph paddingTop={9}>
    Organize styles into reusable component variants, and customize those variants on the fly.
</IntroParagraph>

```jsx
<Heading
    variant="Fancy"
    fontSize="1.875rem"
    letterSpacing="0.1em"
    borderBottom="3px solid"
>
    Hello world
</Heading>
```

<IntroParagraph paddingTop={9}>
    Build a global theme containing design tokens and scales, then use those values to style your app.
</IntroParagraph>

```jsx
<Text
    color="Primary"
    backgroundColor="Secondary"
    padding={2}
    fontSize={4}
>
    Hello world
</Text>
```
